## HTML++ Editor

An Editor to write a **HTML++** code then converts it to normal **HTML** code.

### HTML++ Code Example:

Here some HTML++ code :

```
 html<
        head<
                title<"Test">
        >,
        body<
                h1<"Hello HTML++">
        >

 >

```
That will be converted to :

```
<html>
        <head>
                <title>Test</title>
        </head>

        <body>
                <h1>Hello HTML++</h1>
        </body>
</html>

```

### Tools

- C++ Templates
- FLTK library
